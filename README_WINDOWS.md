# 🗳️ VotingApp — Запуск на Windows

Полная инструкция по развёртыванию и запуску проекта на чистой Windows.

---

## 📋 Что понадобится

| Компонент | Назначение | Ссылка для скачивания |
|-----------|-----------|----------------------|
| **.NET 8 SDK** | Backend (ASP.NET Core Web API) | https://dotnet.microsoft.com/download/dotnet/8.0 |
| **Python 3** | Локальный HTTP-сервер для frontend | https://www.python.org/downloads/ |
| **Docker Desktop** | Контейнеры для MS SQL Server и MailHog | https://www.docker.com/products/docker-desktop/ |
| **Браузер** | Google Chrome / Mozilla Firefox / Edge | — |

> ⚠️ **Важно:** во время установки Docker Desktop убедитесь, что включена опция **WSL 2** (Windows Subsystem for Linux). Docker Desktop предложит установить/обновить WSL2 — соглашайтесь.

---

## 1️⃣ Установка компонентов

### 1.1. Установка .NET 8 SDK

1. Перейдите по ссылке: https://dotnet.microsoft.com/download/dotnet/8.0
2. Скачайте **SDK 8.0.x** для Windows (x64).
3. Запустите установщик и следуйте инструкциям.
4. После установки откройте **PowerShell** или **CMD** и проверьте:
   ```cmd
   dotnet --version
   ```
   Должно отобразиться что-то вроде `8.0.xxx`.

### 1.2. Установка Python 3

1. Перейдите по ссылке: https://www.python.org/downloads/
2. Скачайте последнюю версию Python 3.
3. **Важно:** при установке поставьте галочку **"Add Python to PATH"**.
4. После установки проверьте в PowerShell/CMD:
   ```cmd
   python --version
   ```

### 1.3. Установка Docker Desktop

1. Перейдите по ссылке: https://www.docker.com/products/docker-desktop/
2. Скачайте и установите Docker Desktop для Windows.
3. Перезагрузите компьютер, если потребуется.
4. Запустите Docker Desktop и дождитесь, пока в нижнем левом углу появится надпись **"Engine running"**.

## 2️⃣ Распаковка проекта

1. Скопируйте архив `windarchive.zip` на компьютер.
2. Распакуйте архив в любую папку, например:
   ```
   C:\VotingApp\
   ```
3. Внутри должна быть такая структура:
   ```
   C:\VotingApp\
   ├── voting-backend\
   ├── voting-frontend\
   ├── start.bat
   ├── start.sh
   ├── README_WINDOWS.md
   └── ...
   ```

---

## 3️⃣ Запуск проекта

### Автоматический запуск (рекомендуется)

1. Убедитесь, что **Docker Desktop запущен** (значок в трее, статус "Engine running").
2. Откройте папку с проектом в Проводнике.
3. Дважды кликните по файлу **`start.bat`**.
4. Подождите, пока в консоли появится сообщение **«Все сервисы запущены!»**.

При первом запуске скрипт автоматически:
- скачает и запустит контейнер **MS SQL Server**;
- скачает и запустит контейнер **MailHog**;
- соберёт backend (`dotnet build`), если это необходимо;
- запустит backend на `http://localhost:5000`;
- запустит frontend на `http://localhost:8080`.

> ⏳ Первый запуск может занять несколько минут, так как Docker будет скачивать образы MS SQL Server и MailHog.

### Что делать, если окно `start.bat` сразу закрылось?

Запустите `start.bat` через CMD, чтобы увидеть ошибку:
1. Откройте папку с проектом.
2. В адресной строке Проводника введите `cmd` и нажмите Enter.
3. Введите:
   ```cmd
   start.bat
   ```
4. Смотрите сообщение об ошибке (см. раздел "Возможные проблемы").

## 4️⃣ Открытие приложения

После успешного запуска откройте браузер:

| Сервис | URL |
|--------|-----|
| **Веб-приложение (вход)** | http://localhost:8080/login.html |
| **Документация API (Swagger)** | http://localhost:5000/swagger/index.html |
| **Тестовая почта (MailHog)** | http://localhost:8025 |
| **API** | http://localhost:5000/api |

### Создание администратора

Первый зарегистрировавшийся пользователь автоматически становится администратором.

1. Откройте http://localhost:8080/login.html
2. Переключитесь на вкладку **«Регистрация»**.
3. Заполните форму:
   - **ФИО:** `Администратор`
   - **Email:** `admin@demo.ru`
   - **Пароль:** `password123`
   - **Идентификационный номер:** любой из заранее сгенерированных номеров (если таковые есть) — иначе используйте любой 13-значный номер
4. Нажмите **«Зарегистрироваться»**.
5. После регистрации вы автоматически станете админом и попадёте на главную страницу.

## 5️⃣ Как остановить проект

Закройте окна консоли, которые открыл `start.bat`:
- окно с заголовком **"VotingApp Backend"**;
- окно с заголовком **"VotingApp Frontend"**.

Контейнеры Docker (MSSQL и MailHog) можно остановить в Docker Desktop или командами:
```cmd
docker stop sqlserver
docker stop mailhog
```

## 6️⃣ Пошаговая демонстрация работы

### Сценарий 1: Создание и публикация голосования (админ)

1. Войдите как администратор.
2. Нажмите **«Админка»** в шапке.
3. Вкладка **«Создать голосование»**:
   - **Название:** `Лучший язык программирования`
   - **Описание:** `Какой язык вы предпочитаете?`
   - **Тип:** `Один вариант`
   - **Варианты:** `C#`, `Python`, `JavaScript`
   - **Видимость результатов:** `Всегда видны`
   - **Real-time:** включено
4. Нажмите **«Сохранить черновик»**.
5. Перейдите на вкладку **«Управление»**.
6. Найдите созданное голосование и нажмите **«Опубликовать»**.

### Сценарий 2: Голосование пользователя

1. В другой вкладке браузера откройте http://localhost:8080/login.html
2. Зарегистрируйте обычного пользователя (например, `user@demo.ru` / `password123`).
3. На главной странице выберите опубликованное голосование.
4. Выберите вариант и нажмите **«Проголосовать»**.
5. Результаты отобразятся сразу. Если открыть то же голосование в ещё одной вкладке и проголосовать там, в первой вкладке результат обновится автоматически (SignalR).

### Сценарий 3: Защита от повторного голосования

1. Попробуйте проголосовать повторно тем же пользователем.
2. Система отклонит попытку с сообщением, что вы уже голосовали.

## 7️⃣ Возможные проблемы

| Проблема | Решение |
|----------|---------|
| `dotnet не найден` | Переустановите .NET 8 SDK и перезагрузите компьютер. |
| `python не найден` | При установке Python не была включена опция **Add Python to PATH**. Переустановите Python с этой галочкой. |
| `docker не найден` | Установите Docker Desktop и убедитесь, что он запущен. |
| `Docker Desktop is not running` | Запустите Docker Desktop и дождитесь статуса **"Engine running"**. |
| `Failed to bind to address http://localhost:5000` | Порт 5000 занят. Найдите и закройте процесс, использующий порт 5000. |
| `Порт 8080 занят` | Закройте другие программы, использующие порт 8080. |
| Страница белая / не загружается | Проверьте, что все сервисы запущены: Docker Desktop, backend-окно, frontend-окно. |
| Ошибка при удалении голосования | Убедитесь, что используете последнюю версию `start.bat` — backend должен быть пересобран. |

## 8️⃣ Пересборка backend вручную

Если вы вносили изменения в код backend, пересоберите его:

1. Откройте CMD или PowerShell.
2. Перейдите в папку backend:
   ```cmd
   cd C:\VotingApp\voting-backend\VotingApp
   ```
3. Выполните:
   ```cmd
   dotnet build
   ```
4. Закройте окно **"VotingApp Backend"** и снова запустите `start.bat`.
